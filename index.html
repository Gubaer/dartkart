p. *dartkart* is web cartography framework for the "dart":http://www.dartlang.org programming language.
p. You are running *JavaScript* generated by "dart2js":http://www.dartlang.org/docs/dart2js/. p. If you want to run the "dart":http://www.dartlang.org source natively, you have to install "Dartium":http://www.dartlang.org/dartium/, a version of Google's web browser which includes a Dart virtual machine.
p. You are running *native "Dart":http://www.dartlang.org source* in Dartium. p. This demo site should run in other modern browsers, too. They will run JavaScript generated by "dart2js":http://www.dartlang.org/docs/dart2js/.
